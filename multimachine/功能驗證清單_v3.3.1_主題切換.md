# 功能驗證清單 v3.3.1 — 淺色／深色主題切換

**測試環境：** Cloudflare Pages Preview（feature/multi-machine-type）
**測試帳號：** 訪客體驗模式（登入頁下方「訪客體驗」按鈕）
**測試日期：** ＿＿＿＿  測試人員：＿＿＿＿

---

## 前置：確認預設主題

- [ ] 清除瀏覽器 localStorage（或使用全新無痕視窗）
- [ ] 開啟 App，進入任一頁面
- [ ] 確認預設為**深色模式**（背景為深灰/黑色）

---

## 一、設定頁主題開關

前往「設定」頁面 → 「偏好設定」區塊。

- [ ] 出現「外觀主題」列，顯示月亮／太陽圖示
- [ ] 目前狀態文字顯示「深色模式」
- [ ] 開關為**開啟**狀態（黃色）

### 切換為淺色

- [ ] 點擊開關，主題立即切換為淺色（背景變白）
- [ ] 開關狀態變為關閉（灰色）
- [ ] 狀態文字改為「淺色模式」，圖示變太陽
- [ ] 頁面無需重新整理，切換即時生效

### 切換回深色

- [ ] 再次點擊開關，主題立即切換回深色
- [ ] 開關變回開啟（黃色），文字改回「深色模式」

---

## 二、主題記憶（localStorage）

- [ ] 切換為淺色後，**關閉分頁重新開啟**
- [ ] 確認仍維持淺色模式（不會重置為深色）
- [ ] 切換回深色後，重新開啟，確認維持深色
- [ ] 清除 localStorage 後重新開啟，預設回到深色

---

## 三、各頁面淺色模式外觀檢查

> 切換為淺色後，依序進入各頁面確認可讀性。

### 3-1 Dashboard（首頁）
- [ ] 頂部 header 背景為白/淺色，非黑色
- [ ] 「現金收入」「電支收入」卡片：白底、深色文字
- [ ] 「場地健康狀態」卡片：白底、深色文字
- [ ] 日期篩選按鈕（今日/昨日/…）未選中時為淺灰底
- [ ] 「最近帳務」列表：白底、深色文字

### 3-2 機台監控頁
- [ ] Header 背景為淺色
- [ ] 機台卡片：白底、陰影、深色文字
- [ ] 類型 badge（如 🧸 娃娃機）：淺灰底
- [ ] **上線 badge**：淺綠底、綠色文字（非霓虹綠）
- [ ] **斷線 badge**：淺紅底、紅色文字（非亮紅色）
- [ ] 「均出」數值異常（>800）顯示紅色
- [ ] 「營業額」數值顯示綠色（非霓虹綠）
- [ ] 篩選按鈕（全部／上線／斷線）未選中時為淺灰底

### 3-3 機台詳情 Modal（點擊任一機台卡片）
- [ ] 底部 Modal 背景為白色，非深灰
- [ ] 文字可清楚閱讀（非白字在白底）

### 3-4 財務頁
- [ ] Header 背景為淺色
- [ ] 月份篩選按鈕未選中時為淺灰底
- [ ] 帳單卡片（總刷卡金額）：白底、深色文字
- [ ] 「最近帳務紀錄」列表：白底、深色文字

### 3-5 設定頁
- [ ] 各 section 卡片為白底
- [ ] 次要文字（說明文字、版本號）為灰色，可辨識
- [ ] 「登出帳號」按鈕有紅色文字，白底+紅色框線
- [ ] 底部版權文字可見（非白字在白底）

## 六、登入頁（不受主題影響）

- [ ] 無論主題設為何，登入頁永遠是深色背景（設計上固定不變）

---

## 已知限制（不需回報為 Bug）

- 第一次訪問（無 localStorage）預設為深色
- 主題切換不影響 Login 頁（固定深色）
- 部分系統 UI（瀏覽器原生 date picker）在淺色模式下顏色由瀏覽器決定
