## 二、機台類型篩選列

> 篩選列在狀態篩選（全部／上線／斷線）的下方，只有多種類型時才顯示。

- [ ] 狀態篩選列下方出現第三列類型篩選按鈕
- [ ] 按鈕顯示：「全類型」＋各機型（圖示 + 名稱 + 數量）
- [ ] 點「🧸 娃娃機」→ 只顯示娃娃機（共 3 台），其他機型消失
- [ ] 點「🥚 扭蛋機」→ 只顯示 1 台扭蛋機
- [ ] 點「🐹 打地鼠」→ 只顯示 1 台打地鼠
- [ ] 點「全類型」→ 回到全部 8 台
- [ ] 類型篩選與狀態篩選可以同時作用（例：選「🧸 娃娃機」+「上線」→ 只顯示在線娃娃機）

### 切換場地後類型篩選應正常
- [ ] 場地選「大安店」→ 類型篩選只顯示大安店有的類型
- [ ] 場地選「信義店」→ 類型篩選只顯示信義店有的類型
- [ ] 場地選「全部」→ 類型篩選顯示所有 6 種類型

## 三、日期篩選切換

- [ ] 切換「今日」→ 卡片顯示今日數據，類型 badge 仍正常顯示

- [ ] 切換「即時抄表」→ 卡片正常顯示，類型 badge 仍正常顯示

## 五、Dashboard 營收報表

點擊「營收報表」卡片開啟 Modal。

### 5-1 類型篩選 tab
- [ ] 時間篩選按鈕（24小時內／3天內／7天內／30天內）正常顯示
- [ ] 時間篩選列下方出現機台類型篩選列（「全類型」＋各機型）
- [ ] 點「🧸 娃娃機」→ 熱門機台、異常機台、營收 TOP 3 只顯示娃娃機
- [ ] 點「全類型」→ 回到全部機台
- [ ] 切換時間篩選後，類型篩選狀態維持不變

### 5-2 機台列表類型圖示
- [ ] 熱門機台每筆前面有類型圖示（如「🥚 03號機」）
- [ ] 異常機台每筆前面有類型圖示
- [ ] 營收 TOP 3 每筆前面有類型圖示

### 5-3 異常機台判斷
