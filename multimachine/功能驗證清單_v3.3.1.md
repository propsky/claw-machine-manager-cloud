## 二、機台類型篩選列

> 篩選列在狀態篩選（全部／上線／斷線）的下方，只有多種類型時才顯示。

- [ ] 狀態篩選列下方出現第三列類型篩選按鈕
- [ ] 按鈕顯示：「全類型」＋各機型（圖示 + 名稱 + 數量）
- [ ] 點「🧸 娃娃機」→ 只顯示娃娃機（共 3 台），其他機型消失
- [ ] 點「🥚 扭蛋機」→ 只顯示 1 台扭蛋機
- [ ] 點「🐹 打地鼠」→ 只顯示 1 台打地鼠
- [ ] 點「全類型」→ 回到全部 8 台
- [ ] 類型篩選與狀態篩選可以同時作用（例：選「🧸 娃娃機」+「上線」→ 只顯示在線娃娃機）

### 切換場地後類型篩選應正常
- [ ] 場地選「大安店」→ 類型篩選只顯示大安店有的類型
- [ ] 場地選「信義店」→ 類型篩選只顯示信義店有的類型
- [ ] 場地選「全部」→ 類型篩選顯示所有 6 種類型

## 七、主題切換

前往「設定」→ 「偏好設定」區塊。

### 7-1 開關基本操作
- [ ] 出現「外觀主題」列，顯示月亮／太陽圖示
- [ ] 預設狀態：開關為開啟（黃色），文字顯示「深色模式」
- [ ] 點擊開關 → 主題立即切換為淺色，背景變白
- [ ] 開關變關閉（灰色），文字改為「淺色模式」
- [ ] 頁面無需重新整理，切換即時生效
- [ ] 再次點擊 → 切回深色，開關復原

### 7-2 主題記憶
- [ ] 切換為淺色後，關閉分頁重新開啟 → 仍維持淺色
- [ ] 切換回深色後，重新開啟 → 仍維持深色
- [ ] 清除 localStorage 後重新開啟 → 預設回到**深色**

### 7-3 淺色模式各頁外觀

切換為淺色後，依序確認各頁面可讀性。

**Dashboard**
- [ ] 頂部 header 為淺色背景
- [ ] 「現金收入」「電支收入」卡片：白底、深色文字
- [ ] 「場地健康狀態」卡片：白底、深色文字
- [ ] 日期篩選按鈕未選中為淺灰底

**機台監控頁**
- [ ] 機台卡片：白底、深色文字
- [ ] 類型 badge（🧸 娃娃機 等）：淺灰底、可辨識
- [ ] **上線 badge**：淺綠底＋綠色文字（非霓虹綠）
- [ ] **斷線 badge**：淺紅底＋紅色文字（非亮紅色）
- [ ] 「營業額」數值：綠色（可辨識，非霓虹綠）
- [ ] 點擊機台卡片開啟 Modal → Modal 為白底、深色文字

**財務頁**
- [ ] 帳單卡片：白底、深色文字
- [ ] 篩選按鈕未選中：淺灰底
- [ ] 「最近帳務紀錄」列表：白底、深色文字

**設定頁**

**底部 Sheet（財務頁 → 自訂範圍）**

**場地選擇器下拉（點擊頂部場地名稱）**

- 販賣機在「即時抄表」模式下營業額顯示 $0（因為沒有固定單價，需後端支援）

- 機台類型目前由 config 靜態設定，後台設定介面待後端完成後補上
- 第一次訪問（無 localStorage）主題預設為深色
- 登入頁主題固定深色，不跟隨設定
